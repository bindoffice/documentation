docker 安装¶

0. 环境要求¶

最低1G内存 1核CPU linux 64位系统

1. 配置DNS¶

在域名服务提供商或者DNS解析服务中配置域名的一条A记录指向服务器的IP

例如

主机名

记录类型

记录值

meet

A

x.x.x.x

  • 主机名可以为任意二级域名前缀，这里我们使用meet

  • example.com 是你的域名

  • x.x.x.x 是服务器的公网IP地址

  • 访问地址是 https://meet.example.com/meet

2. 安装 docker, git¶

# If already installed, skip

mkdir pkgs && cd pkgs
wget 'https://download.docker.com/linux/static/stable/x86_64/docker-26.1.4.tgz'
tar xf docker-26.1.4.tgz;
chmod +x docker/*
sudo cp docker/* /usr/local/bin/

curl -SL https://github.com/docker/compose/releases/download/v2.27.1/docker-compose-linux-x86_64 -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose

yum install git # centos
apt install git  # ubuntu/debian

# start docker daemon
sudo dockerd &

3. 安装¶

# github 国外用户使用github更快
git clone https://github.com/BindOffice/bind-docker.git

# gitee 国内用户使用gitee更快
git clone https://gitee.com/hedwi/bind-docker.git

cd bind-docker/meet
cp env.example .env

# Modify the .env configuration

make

  • Configuration file example

4. 自动生成免费SSL证书¶

make cert

5. 重启nginx¶

make down && make
