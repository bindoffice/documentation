鉴权¶

1. 鉴权类型¶

  • none: 无鉴权

  • ldap: LDAP 鉴权

2. 如何配置鉴权¶

在 .env 文件中设置 MEET_AUTH_TYPE

3. 如何配置 LDAP 鉴权¶

在 .env 文件中设置以下变量

MEET_LDAP_ENABLED="true" # 是否启用 LDAP 鉴权 MEET_LDAP_ADDR="ldap://127.0.0.1:42389" # LDAP 服务器地址 MEET_LDAP_BASE_DN="dc=hedwi,dc=com" # LDAP 基础 DN

4. 如何在移动应用中使用鉴权¶

  • LDAP 鉴权启用，移动应用将使用 LDAP 鉴权

    密码 = LDAP 密码 + " " + 会议密码

  • LDAP 鉴权禁用，移动应用将使用会议密码，如果会议密码未设置，则留空

    密码 = 会议密码或空白
