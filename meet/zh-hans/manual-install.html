手动部署¶

0. 环境要求¶

最低1G内存 1核CPU linux 64位系统

1. 安装依赖¶

centos / redhat

    yum install -y nginx supervisor redis

debian / ubuntu

    apt install -y nginx supervisor redis

2. 安装 meet server¶

export HEDWI_ROOT=/data/www


mkdir -p $HEDWI_ROOT/{bind-meetserver,redis}/logs;
cd $HEDWI_ROOT ;
wget 'https://download.hedwi.com/mail/linux/amd64/bind-meetserver.tar.gz'
tar xf bind-meetserver.tar.gz;

wget 'https://download.hedwi.com/mail/bind-meetserver-conf.tar.gz';
tar xf bind-meetserver-conf.tar.gz;
cp bind-meetserver-conf/bind-meetserver.conf .;
cp bind-meetserver-conf/config.yaml.example bind-meetserver/config.yaml;
# Modify the domain name in the configuration file.

supervisord -c /data/www/bind-meetserver.conf;

配置文件示例

3. nginx 和 ssl 证书配置¶

mkdir -p $HEDWI_ROOT/ssls/;
cd $HEDWI_ROOT/ssls;
mkdir example.com; //Domain certificate folder, here is the corresponding domain name
cd !$;
touch cert.crt; //Add certificate
touch cert.key; //Add certificate private key

Open /etc/nginx/conf.d/meet.conf and fill in the content according to the example file (replace the domain name and port).

Nginx 配置文件示例

4. 重启 nginx¶

# Test nginx configuration
nginx -t

# Restart nginx
nginx -s reload

#If nginx is not started, start nginx
nginx
