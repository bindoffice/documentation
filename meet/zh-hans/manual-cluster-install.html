手动集群部署¶

docker 安装¶

curl -O 'https://hedwi.com/cluster' | sh

开始¶

docker compose up -d
  • 防火墙配置

    开放25, 465, 993, 443端口用于外部访问。邮件服务不需要nginx。如果使用nginx，需要修改配置文件中的端口。

  • 创建管理员账户

    访问 https://127.0.0.1:40008 (可以是根域名或任何二级域名，只要DNS记录配置正确)。注册管理员账户。第一个账户是管理员账户。使用管理员账户添加域名并配置DNS记录。
